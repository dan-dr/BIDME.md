Open Source

Monetize your
GitHub README

BidMe lets anyone bid on banner space in your repository's README. The highest bidder's ad goes live automatically via GitHub Actions.

Get Started
1

Install

Add the BidMe config and GitHub Action to your repo. Takes under a minute.

2

Bids Come In

Anyone can submit a bid by opening an issue. The bot validates and tracks every bid.

3

Get Paid

The highest bidder's banner goes live in your README. You collect payment via Polar.sh.

Why BidMe?

Open-source maintainers deserve to get paid. BidMe turns your most visible asset — your README — into a revenue stream, without selling out your project.

BidMe is open source under the MIT License. Built with Bun, GitHub Actions & GitHub Pages.
