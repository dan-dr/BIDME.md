BidMe

Auction-based README banner sponsorship for GitHub repos

How it works

Repo owners run bidme init to set up periodic banner auctions. Sponsors bid by commenting on the auction issue with a YAML block. The highest bidder's banner gets displayed in the README until the next period.

For sponsors

Find a repo using BidMe, comment on the open auction issue with your bid amount, banner image URL, and destination link. Link a payment method via Stripe to activate your bid.
